# omega-calculus

Requirements: https://www.haskellstack.org/

To run:
  * stack build
  * stack exec omega-calculus-exe


Has an example from the paper hardcoded. Modify Main.hs to make it run a different example.

This repository is also uploaded anonymously to anonymous.4open.science. To make the code easier to download from that source, a ZIP file of the repository has also been uploaded to https://omegacausal.s3.amazonaws.com/omega-calculus-master.zip .
